Thuộc tính CSS trong Flexbox: - display: flex | inline-flex => quyết định có sử dụng layout flexbox hay ko? - flex-direction: row | column => thay đổi phương hướng của main axis - flex-wrap: nowrap | wrap | wrap-reserve => wrap: xuống dòng (ko xuống | xuống | nhảy lên trên) - flex-basic: => set kích thước cho main size - justify-content: flex-start | flex-end | center | space-between | space-around => căn flex-item theo hướng (trái/phải/giữa/trên/xuống dưới) - justify-self: flex-start | flex-end | center - align-content: flex-start | flex-end | center (set cho flex container) - align-self: flex-start | flex-end | center (set cho flex item) - flex-grow: => thay đổi kích thước main size của flex item => làm lớn lên - flex-shrink: => thay đổi kích thước main size của flex item => thu nhỏ lại - flex: => viết gộp của flex-baisc flex-grow flex-shrink - order: => quyết định thứ tự flex-item (cái nào trước cái nào sau) - flex-flow => cú pháp short-hand của flex-wrap and flex-direction
