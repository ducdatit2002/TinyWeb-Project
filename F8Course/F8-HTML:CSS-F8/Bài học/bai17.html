cách dùng hàm giả: 1. hover: dùng để khi di chuột đến chữ đó thì hiện màu,... h1:hover{ color: red; } 2. active: bấm và giữa chuột thì hiện màu,... cách dùng giống hover 3. first-child: dùng hàm để thay đổi màu của thằng đầu tiên cách dùng giống hover 4. last-child: dùng hàm để thay đổi màu của thằng cuối cách dùng giống hover
